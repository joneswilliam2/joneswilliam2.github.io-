# joneswilliam2.github.io

Academic website for Will Jones, M.A. candidate in Applied Economics at the University of Houston.

## Structure

```
index.html        — About / Home
research.html     — Research papers and works in progress
cv.html           — Curriculum vitae
contact.html      — Contact information
style.css         — Stylesheet
cv.pdf            — CV (add your own PDF)
permian_basin.pdf — Working paper PDF (add when ready)
```

## Setup

1. Push all files to a repository named `joneswilliam2.github.io`
2. Go to Settings → Pages → Source: Deploy from branch → main → / (root)
3. Site goes live at https://joneswilliam2.github.io within a few minutes

## To update

Edit the HTML files directly and push. No build step required.
